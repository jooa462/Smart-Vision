Features

Hardware-Software Integration

Smart Vision integrates the ESP32-CAM hardware module with a powerful web system built using Next.js and Node.js. This seamless connection enables real-time scanning of QR codes and immediate data storage in MongoDB, ensuring fast and accurate product tracking.

More

User & Product Management

The system includes a comprehensive dashboard that allows administrators to manage users with different access roles, as well as handle product inventory efficiently. Adding, editing, or deleting entries is simple and scalable.

More

Energy-Efficient Design

Smart Vision runs on a rechargeable Li-Po battery, making it ideal for mobile or power-limited environments. The ESP32 module ensures minimal energy consumption, allowing the system to operate for extended periods without frequent recharging.

How It Works ?

Hardware-Software Integration

Overview: The ESP32-CAM module captures QR codes from products. Once scanned, the data is transmitted via Wi-Fi to a backend server powered by Node.js and Express. This server processes the information and stores it in a MongoDB database, ensuring efficient and real-time data handling.

User and Product Management

Overview: An intuitive dashboard, built with Next.js, allows administrators to manage users with varying access levels and oversee product inventories. Features include adding, editing, and deleting user profiles and product details, all synchronized with the MongoDB database.

Energy-Efficient and Portable Design

Overview: The system operates on a rechargeable Li-Po battery, ensuring portability and flexibility in various environments. The ESP32 module's low power consumption extends operational time, making it suitable for locations without constant power sources.
